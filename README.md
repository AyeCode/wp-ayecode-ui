# AyeCode UI

A Bootstrap UI library for WordPress plugins and themes.

## Internal Use Only

This library is for **internal AyeCode Ltd use only** and is designed to be used across AyeCode products.

## Version 3.0+

Version 3.0 and above introduces the following breaking changes:

- **Dark/Light Mode Support**: Full support for dark and light mode theming
- **Bootstrap 5.3+ Only**: Earlier versions of Bootstrap are no longer supported
- **Minimum Requirements**: PHP 7.4+ and WordPress 5.0+

## Installation

This library is typically bundled with AyeCode plugins and themes. It uses a version detection system to ensure only the latest version is loaded when multiple AyeCode products are active.
